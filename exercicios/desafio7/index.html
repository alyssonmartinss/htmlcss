Tags em HTML

Em html podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag utilizar em cada caso.

Paragrafos
Use a tag p
Titulos
Use as tags h1 ate a h6
Imagens
Use a tag img
abreviaçoes
Use a tag abbr
Codigo fonte
Use a tag code
Listas numeradas
use as tags ul e ol
Lista com demarcações
Use exatamente as tags que foram usadas aqui dt e dd
